# AI赋能教学实操培训 · 微学习平台

## 快速开始

### 方式一：纯前端模式（最简单）

双击 `student.html` 即可在浏览器中打开使用。

- 数据存储在浏览器本地（localStorage），刷新不丢失
- 支持注册/登录、学习进度追踪、积分/徽章/等级
- 排行榜功能需要后端支持（离线模式下不可用）

### 方式二：完整模式（推荐培训现场使用）

启动后端服务器，支持云端认证、跨设备登录、实时排行榜。

```bash
# 1. 安装依赖
pip3 install -r server/requirements.txt

# 2. 启动服务器
cd /workspace/training-site
python3 -m uvicorn server.app:app --host 0.0.0.0 --port 8000

# 3. 打开浏览器访问
# 本机访问：http://localhost:8000
# 局域网其他设备访问：http://<本机IP>:8000
```

## 功能说明

### 认证系统
- 手机号 + 密码注册/登录
- 在线模式：JWT 令牌，跨设备登录
- 离线模式：本地存储，单设备使用

### 学习进度
- 13 个学习关卡（5 个认知关 + 8 个实操关），完成即记录
- 进度自动持久化（在线存服务端，离线存 localStorage）
- 刷新页面或重新登录后进度保留

### 游戏化激励

**积分规则：**

| 行为 | 积分 |
|---|---|
| 每日首次登录 | +5 |
| 选择任务 | +20 |
| 完成单关（K1-K5 / P1-P8） | +10/关 |
| 完成认知关（K1-K5全齐） | +30 bonus |
| 完成实操关（P1-P8全齐） | +80 bonus |
| 全部13关完成 | +150 bonus |

**等级体系（8级）：**

> 单次培训理论积分上限约 415 分，可摸到 Lv.7；Lv.8 留给跨次培训成长。

| 等级 | 名称 | 门槛 |
|---|---|---|
| Lv.1 | AI新兵 | 0 |
| Lv.2 | 提示词学徒 | 40 |
| Lv.3 | 教案能手 | 90 |
| Lv.4 | 技能行家 | 150 |
| Lv.5 | 分层达人 | 220 |
| Lv.6 | 审核专家 | 290 |
| Lv.7 | AI教学先锋 | 360 |
| Lv.8 | 微学习大师 | 500+ |

**徽章体系（6枚）：**

| 图标 | 名称 | 触发条件 |
|---|---|---|
| 🚀 | 启程者 | 首次登录 |
| 🎯 | 抉择者 | 选择任意任务 |
| ⚡ | 认知达人 | 完成5个认知关 |
| 🏁 | 实战先锋 | 完成8个实操关 |
| 🏆 | 满分通关 | 13关全部完成 |
| 📚 | 探索者 | 累计复制3次提示词 |

### 排行榜
- 在线模式下显示全体学员积分排名
- 按积分降序排列，Top 20
- 当前用户行高亮显示
- 离线模式下显示提示信息

## 在线/离线模式自动切换

前端启动时会自动探测后端服务器：
1. 先尝试同源地址（如果通过 http 访问）
2. 再尝试 `http://localhost:8000`
3. 2秒内无响应则进入离线模式

顶部栏显示当前模式：
- 🟢 云端已连接 = 在线模式
- ⚪ 本地模式 = 离线模式

## API 接口

| 方法 | 路径 | 说明 |
|---|---|---|
| GET | `/api/health` | 健康检查 |
| POST | `/api/auth/register` | 注册 |
| POST | `/api/auth/login` | 登录 |
| GET | `/api/auth/me` | 获取当前用户 |
| GET | `/api/progress` | 获取学习进度 |
| PUT | `/api/progress/step` | 更新步骤完成状态 |
| PUT | `/api/progress/task` | 选择任务 |
| PUT | `/api/progress/explorer` | 授予"探索者"徽章（复制3次提示词触发） |
| GET | `/api/leaderboard` | 获取排行榜 |

## 培训现场使用建议

1. **培训前**：讲师在主机上启动后端服务器，确保局域网可用
2. **学员访问**：学员用手机或电脑浏览器访问 `http://<讲师IP>:8000`
3. **注册登录**：学员用手机号快速注册（密码至少6位）
4. **学习过程**：跟着讲师推进13个关卡，每完成一关点击"完成"获得积分
5. **排行榜**：学员可随时查看自己在全班的排名
6. **培训后**：学员进度保存在服务端，下次登录可继续

## 技术栈

- 前端：纯 HTML/CSS/JavaScript，无构建步骤，无外部依赖
- 后端：Python FastAPI + SQLite + JWT + bcrypt
- 数据库：SQLite（零配置，自动创建）
