入门：两个关键能力

怎么和AI说话，决定了它能帮你多少

Prompt · 你怎么问
本质：问得越准确，AI答得越好
类比：就像你给学生出题——题目质量决定答案质量
关键技巧
明确角色 给定结构 提供范例 设约束条件
"你是一位有10年教龄的初中语文老师，请以表格形式输出《背影》的3个分层教学目标"
Context · 你给什么背景
本质：给AI越多背景材料，答案越精准
类比：给阅读材料再提问 vs 直接提问
可以放进去的
课程大纲 学生情况 学校要求 往年试卷
把课程大纲 + 学生情况 + 学校要求一起给AI，让它帮你做整个学期的教学计划

工具策略

不是换工具，是知道什么时候用哪个

封闭工具（讯飞/小鹿）
标准化批改、数据采集
课堂平板互动
适合：日常作业、课堂练习
短板：不够灵活，资源预设，复习课/个性化场景覆盖不了
通用AI（豆包 / 元宝）
按需即时生成内容
灵活定制教学方案
适合：复习课、分层教学、个性化辅导
短板：需要你会用——但这正是今天在学的
聊天对话 · 备课答疑
豆包
备课助手、出题、答疑、创意激发。免费、上手快，建议作为第一个尝试的工具。
长文档处理 · 写作办公
腾讯元宝
读课标/长文档、总结要点、生成教案PPT草稿、整理会议纪要。
⚠ 重要提醒： 含学生姓名、成绩、个人信息的材料，上传前先脱敏（替换为编号），不要把原始数据直接传给通用AI。

封闭工具做"标准化"的批改采集，通用AI做"个性化"的即时生成

给校长的建议

工具已经到位
下一步是让人和流程把它用深、用透

已有基础，可继续投入
讯飞智批和小鹿平板已经铺开——下一步是配套的教学流程和带头人，把工具的价值充分释放
培养一个"教学+AI"带头人
不一定要新增编制，可先从教研组长、骨干教师中培养1-2位兼职的"AI教学带头人"，负责把工具能力和教学需求对接
从3位老师跑通一个最小闭环
找最愿意试的3位老师，各带1个班级、跑1个学期，用批改复核时间 + 单元测成绩两个口径验证效果，再向全校推广
推AI赋能，关键是建制度、建文化、给带头人支持。
